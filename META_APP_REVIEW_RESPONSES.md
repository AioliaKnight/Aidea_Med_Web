**我們僅使用 oEmbed Read 功能，具體用途如下：**

- **API 端點：** Facebook oEmbed API, Instagram oEmbed API
- **使用目的：** 在醫療行銷網站上嵌入公開的社群媒體內容

- 我們**不使用 Facebook 登入**功能
